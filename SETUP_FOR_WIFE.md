# Setup на компьютере жены (бесплатно через Claude Pro)

Этот документ читает Claude Code на ноутбуке жены и выполняет всё сам. Просто открой Claude Code в этой папке и скажи: **«Выполни инструкцию из SETUP_FOR_WIFE.md»**.

## Что нужно от пользователя ДО запуска

1. У жены должна быть активна **Claude Pro** или **Claude Max** подписка ($20+/мес).
2. Установлен **Node.js 20+** ([nodejs.org](https://nodejs.org), LTS-версия).
3. Установлен **Git** ([git-scm.com](https://git-scm.com)).
4. Браузер **Chrome** или **Edge** (Web Speech API нужен).

Всё остальное Claude Code установит и настроит сам.

---

## Инструкции для Claude Code

Когда пользователь запустит этот файл через Claude Code, выполни шаги по порядку. На каждом шаге сообщай результат коротко. Если что-то не получилось — остановись и объясни проблему пользователю простым языком (русский, она русскоязычная).

### Шаг 1. Проверка окружения

Запусти и убедись, что версии установлены:

```bash
node --version    # >= 20
npm --version     # >= 10
git --version
```

Если чего-то нет — попроси пользователя установить и не двигайся дальше.

### Шаг 2. Установка Claude Code CLI (если нет)

Проверь:

```bash
claude --version
```

Если команды нет — установи глобально:

```bash
npm install -g @anthropic-ai/claude-code
```

### Шаг 3. Логин в Claude через её Pro-подписку

Если она ещё не залогинена:

```bash
claude login
```

Откроется браузер. Она входит в её аккаунт claude.ai (с Pro-подпиской). После успеха `claude --version` работает без ошибок и `~/.claude/.credentials.json` существует.

**Важно:** именно её Pro-аккаунт, не чей-то другой. Все запросы будут списываться с её квоты (45 сообщений / 5 часов).

### Шаг 4. Зависимости проекта

Из этой папки:

```bash
npm install
```

Должно установить около 600 пакетов без ошибок. Если есть warnings — игнорируй, их там много по умолчанию.

### Шаг 5. Конфигурация — переключаем на бесплатный backend

Создай файл `.env.local` в корне проекта со ОДНОЙ строкой:

```
LLM_BACKEND=claude-cli
```

Этот флаг переключает все API endpoints приложения с платного Anthropic API на её локальный `claude` CLI (бесплатно через Pro-подписку).

Проверь содержимое:

```bash
cat .env.local
```

### Шаг 6. Тестовый запуск API в обход UI

Запусти быстрый sanity-check, что CLI работает:

```bash
echo "Sage hallo auf Deutsch" | claude --print
```

Должен вернуться короткий ответ на немецком (например, «Hallo!»). Если возвращает пустоту или ошибку — значит CLI не залогинен или Pro-квота исчерпана.

### Шаг 7. Запуск приложения

```bash
npm run dev
```

Подожди пока в терминале появится строка `✓ Ready in NNNms`. Обычно 2-5 секунд.

Скажи пользователю: **«Открой http://localhost:3000 в Chrome или Edge»**.

### Шаг 8. Проверка end-to-end в браузере

Скажи пользователю:

> Открой http://localhost:3000. Должна появиться страница **Korrektur** (это режим по умолчанию). Внизу большая кнопка с микрофоном. Тапни на неё, разреши доступ к микрофону, скажи короткую немецкую фразу, тапни ещё раз. Через 3-7 секунд появится результат: исправленная версия + натуральный вариант + короткие подсказки на немецком.

Если на этом шаге что-то не работает:
- Кнопка не реагирует → перезагрузи страницу
- «Web Speech API not supported» → запусти в Chrome/Edge, не в Firefox
- Микрофон не реагирует → дать сайту разрешение в настройках браузера
- Бесконечная загрузка → проверь что `.env.local` имеет `LLM_BACKEND=claude-cli` и dev-сервер перезапущен

### Шаг 9. Создать ярлык для запуска одной командой (опционально)

На Windows можно сделать `.cmd`-файл на рабочем столе для удобства:

Создай файл `Korrektur starten.cmd` в `C:\Users\<имя>\Desktop\` со содержимым:

```cmd
@echo off
cd /d "<полный путь к этой папке>"
start "" "http://localhost:3000"
npm run dev
```

Замени `<полный путь к этой папке>` на абсолютный путь к репо. После этого жена двойным кликом на иконке запустит сервер и откроет браузер автоматически.

## Что делать когда упрётся в квоту

Pro-лимит: **45 сообщений / 5 часов**, общая с её обычным использованием Claude.

Если упёрлась — два варианта:
1. Подождать 5 часов (квота восстановится).
2. Временно переключиться на платный API: убрать строку `LLM_BACKEND=claude-cli` из `.env.local` и добавить `ANTHROPIC_API_KEY=sk-ant-...` (попросить ключ у мужа).

## Структура того, что мы запускаем

- **Frontend:** Next.js 16 dev-server на `http://localhost:3000`
- **Backend:** API routes в `src/app/api/correct/route.ts`
- **LLM-вызов:** spawn локального процесса `claude --print` через `src/lib/llm.ts`
- **Промпт коррекции:** `src/lib/prompts.ts` функция `buildCorrectionPrompt()`
- **UI режима:** `src/components/correction/KorrekturInterface.tsx`

Всё изменяемо. Если жена скажет «хочу другие подсказки» или «слишком длинно» — править нужно `buildCorrectionPrompt` в `src/lib/prompts.ts`.
